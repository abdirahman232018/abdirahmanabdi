Multiplayer Pong Game

This was a project I started back in summer 2021 when I first learned python. This was a capstone project instructed by Angela Yu (Udemy instructor).

The game was played by my friends among themselves , it was entertaining and interesting to watch them play.

They had the ability to see the game scoreboard change on time and see who is winning or losing.

This was the most fulfilling as beginner to python by then.

I since made tens of long projects which are available on github

Technologies:
  • - Python3 OOP
  • - Tkinter library
  • - Turtle library
  • - Pycharm IDE
